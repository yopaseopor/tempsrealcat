Fonts de Dades

Aquesta pàgina detalla totes les fonts de dades utilitzades en Temps Real Cat per proporcionar informació en temps real sobre transports i trànsit a Catalunya.

Llegenda

API API oficial
RSS Canal RSS
XML Dades XML
JSON Dades JSON
GBFS Sistema de bicicletes compartides

Arquitectura de l'API

Totes les dades es consulten a través d'una capa d'API proxy a Vercel per evitar problemes de CORS i permetre l'accés des del navegador. Els endpoints estan disponibles a:

https://tempsrealcat.vercel.app/api/[nom-del-servei]

Trànsit

DGT - Dirección General de Tráfico
XML DATEX2
Dades d'incidents de trànsit en temps real de la DGT espanyola en format DATEX2. Inclou accidents, obres, congestions, talls de carretera i altres incidents.
https://infocar.dgt.es/datex2.xml
https://infocar.dgt.es/xml.xml
https://www.dgt.es/incidencias/incidencias.xml
https://dgt.es/incidencias.xml
Actualització: En temps real
GENCAT - Generalitat de Catalunya (RSS)
RSS XML
Canal RSS d'incidents de trànsit del Servei Català de Trànsit. Proporciona informació actualitzada sobre incidents a les carreteres de Catalunya.
https://www.gencat.cat/transit/rss/ca/
Actualització: En temps real
GENCAT - Generalitat de Catalunya (GML)
XML GML
Dades geoespacials d'incidents de trànsit en format GML (Geography Markup Language) de la Generalitat de Catalunya. Inclou informació detallada amb coordenades geogràfiques.
https://intranet.gencat.cat/economia/coneixementai/governobert/dades_obertes/dades_obertes_informacio_transit.xml
Actualització: En temps real

Transport Ferroviari

RENFE - Red Nacional de Ferrocarriles Españoles
JSON GTFS-RT
Dades en temps real de la posició dels trens RENFE a través del protocol GTFS-Realtime. Inclou informació de posició i estat dels serveis ferroviaris.
https://gtfsrt.renfe.com/vehicle_positions.json
Actualització: En temps real
FGC - Ferrocarrils de la Generalitat de Catalunya
JSON API
Dades de posicionament dels trens FGC a través de l'API de dades obertes de FGC. Proporciona informació actualitzada sobre la flota de trens FGC.
https://dadesobertes.fgc.cat/api/explore/v2.1/catalog/datasets/posicionament-dels-trens/records?limit=100
Actualització: En temps real

Transport Urbà

TMB - Transports Metropolitans de Barcelona
JSON API
Dades del servei d'autobusos de TMB a través de la seva API iTransit. Inclou informació de parades, horaris i temps d'espera.
https://api.tmb.cat/v1/itransit/bus/parades/108
Actualització: En temps real
TMB - Metro de Barcelona
JSON API
Dades de les estacions i línies de metro de TMB. Proporciona informació sobre la xarxa de metro i l'estat del servei.
https://api.tmb.cat/v1/itransit/metro/estacions
Actualització: Periòdica

Combustible

Ministerio para la Transición Ecológica - España
JSON API
Dades oficials de preus de combustibles a les estacions de servei espanyoles proporcionades pel Ministeri per a la Transició Ecològica i el Repte Demogràfic. Inclou preus actualitzats de gasolina 95, dièsel i altres combustibles.
https://sedeaplicaciones.minetur.gob.es/ServiciosRESTCarburantes/PreciosCarburantes/EstacionesTerrestres/
Actualització: Diària
OpenStreetMap - Estacions de Servei
API Overpass
Dades geogràfiques d'estacions de servei i punts de recàrrega d'OpenStreetMap obtingudes a través de l'API Overpass. Proporciona informació sobre ubicació, serveis disponibles i tipus de combustibles.
https://overpass-api.de/api/interpreter
https://overpass.kumi.systems/api/interpreter
https://overpass.osm.ch/api/interpreter
https://overpass.openstreetmap.fr/api/interpreter
Actualització: Segons la demanda

Mobilitat Sostenible

Bicing - Barcelona
JSON GBFS
Dades del sistema de bicicletes compartides Bicing de Barcelona a través del protocol GBFS (General Bikeshare Feed Specification). Inclou estat de les estacions, disponibilitat de bicicletes i places lliures.
https://barcelona.publicbikesystem.net/customer/gbfs/v2/en/station_status
Actualització: En temps real

API OSM
Dades cartogràfiques i de POI (Points of Interest) d'OpenStreetMap. Utilitzat per a la geocodificació i visualització de mapes base.

https://nominatim.openstreetmap.org/
https://overpass-api.de/

Wikidata
API SPARQL
Dades estructurades de Wikidata per obtenir informació contextual sobre ubicacions, entitats i relacions geogràfiques.
https://query.wikidata.org/sparql
https://www.wikidata.org/
Actualització: Contínua
GraphHopper
API
Servei de routing i planificació de rutes utilitzat per calcular rutes a peu, en bicicleta i transport públic.
https://graphhopper.com/maps/
Actualització: En temps real
Mobility Database
API GTFS
Base de dades global de conjunts de dades GTFS de transports públics de tot el món. Utilitzat per explorar i descarregar dades de transports.
https://mobilitydatabase.org/
Actualització: Regular

Privacitat i Seguretat

  • Totes les cridades a les APIs es fan a través d'un proxy per evitar exposar claus d'API
  • No es recullen dades personals dels usuaris
  • Les dades de localització només es processen localment al navegador
  • Les peticions a les APIs externes tenen límits per evitar sobrecàrrega

Freqüència d'Actualització

Quant a Temps Real Cat

Sobre el Projecte

Aquest és un mapa de la nostra ciutat creat a partir d'un projecte de l'associació sense ànim de lucre la Konfraria de la Vila del Pingüí de la Palma (de Cervelló), que a més a més ha participat activament en el grup d'OSM en català així com en la incorporació de dades al mapa de la Palma de Cervelló al projecte OpenStreetMap.

Trobareu un resum de les activitats més rellevants a la pàgina de l'usuari KonfrareAlbert, i també informació sobre la importació del cadastre a la pàgina de la Palma de Cervelló al wiki d'OSM. Trobareu més informació sobre la Konfraria, i el nostre contacte en aquest enllaç.

Codi del Projecte

Codi del projecte:
OpenLocalMap (llicència GPL) a GitHub.

Aquest lloc web és un fork del projecte osm-pois de Manuel Kaufmann, del que n'hem aprofitat gran part del codi (POIs al blog d'Humitos, codi d'osm-pois a GitHub amb llicència GPL, Twitter: @reydelhumo).
